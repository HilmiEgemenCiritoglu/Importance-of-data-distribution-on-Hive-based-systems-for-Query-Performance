# Importance of Data Distribution on Hive-based Systems for Query Performance: An Experimental Study

This repository includes customised configs for the TPC-H_on_Hive benchmarks which is build to run with different execution engine. The tool is used during tests for "Importance of Data Distribution on Hive-based Systems for Query Performance: An Experimental Study"

For TPC-H_on_Hive benchmark, the following link can be used: [TPC-H-Hive](https://github.com/rxin/TPC-H-Hive).
